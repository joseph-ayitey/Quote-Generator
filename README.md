# Quote-Generator
Module-Decomposition 

A full-stack quote generator application built to understand how frontend and backend systems communicate through APIs and how separate services are deployed independently.

---

# 1. Why I Built It

This project was built to learn:

• How frontend and backend systems communicate
• How APIs work
• How separate services are deployed
• Software decomposition and planning
• Full-stack application structure

The project also helped me understand how multiple pieces of software interact together in a real-world application.

---

# 2. Features

• Random quote generator
• Add new quotes
• Delete quotes
• Frontend and backend API communication
• Separate frontend and backend deployment

---

# 3. Technologies Used

## Backend

# 5. Software Engineering Concepts Learned

| Concept | Description |
|---|---|
| Planning | Understanding the project before coding |
| Decomposition | Breaking the project into smaller tasks |
| Data Structures | Organising quote data between frontend and backend |
| Dependencies | Understanding how frontend depends on backend APIs |
| Requirements | Defining application functionality before development |

---

# 6. Suggested Build Order

1. Build frontend with dummy data
2. Build backend API
3. Connect frontend to backend
4. Add add/delete functionality
5. Deploy backend
6. Deploy frontend
7. Submit project links
